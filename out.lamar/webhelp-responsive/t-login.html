How To Log On to the Computers In Class

This task will teach you the basics about logging into the provided Macintosh computers. Typically these are available for use to Virginia Tech students only, but AirClassroom has provided you with a default login to use across all computers.

Across the room, there are about 25 computers available to you, the user.

Classroom Usage RulesPlease see Classroom Rules Page for best practices with regard to the machines.
  1. The Figure above shows the mac computers that you will be using. Press any key to illuminate the lock screen.
  2. Observe the whiteboard at the front of the classroom
  3. To ensure privacy, the username and password will be written down on the whiteboard for each individual class before you enter. These are the values that you will enter in to the computer to unlock them
    In the event that all of the users are Virginia Tech students for the game night, please let them know they should use their PID and password (the same login that they use for canvas.)
  4. The lock screen will have two entry boxes in white, one that says PID and one that says password. See Figure 2 for details
  5. Enter both values, and press the Enter key
