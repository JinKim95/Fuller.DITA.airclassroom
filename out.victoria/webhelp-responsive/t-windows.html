How To Run Windows on the Provided Mac Computers

All of the Macintosh computers in the room also come with the capability to run Windows OS as well as the standard Mac OS. Many programs and functionalities are run better on Windows, so if you need to access that operating system, you are free to do so.

  1. Boot up the computer by clicking any key to cancel "Sleep Mode"
  2. You will be presented with a screen that gives you two "boot" options, if you wish to choose Windows, click the option entitled "Bootcamp" on the right side of the screen with your mouse See Figure 7 for Image of screen.
    The hard drive on the left is Macintosh HD, if you wish to boot Mac OS like normal, select this option.
  4. Type in the login information provided on the whiteboard in the front of the room
  7. Begin working within your chosen operating system If you have any issues logging on, it may be because you have attempted login too many times. If this occurs, look to the rear of the physical computer screen. On the back, in the bottom right hand corner of the screen, there is a small, round physical power button. Hold this button down, and the computer will restart. See Figure 6 for image of the button.
